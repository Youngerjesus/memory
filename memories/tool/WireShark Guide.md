## What is Wireshark

- 와이어샤크는 network packet 분석 도구

### Features

- tcpdump/WinDump 를 통해서 패킷 안의 데이터를 보는것 가능
- 네트워크 인터페이스로 들어가는 패킷 캡처 가능
- text 파일 형태의 데이터 (hex dump 를 가지고 있는.) 를 import 해서 패킷 형태로 보는것 가능.
- 프로토콜 정보 보는것 가능
- 캡쳐된 패킷 저장 가능
- 특정한 파일 포맷 형태로 패킷 export 가능
- 패킷 필터링 가능
- 패킷 검색 가능
- 통계 기능 만들기 가능
